# Image Resizer
An image resizer app for desktop.

## Download
MacOS

Windows

## Development
```sh
# 1. Download the repo.
git clone https://github.com/ozgrozer/electron-image-resizer.git

# 2. Change directory.
cd electron-image-resizer

# 3. Install dependencies (with npm or yarn).
npm install
yarn install

# 4. Start development.
yarn start

# 5. Build the app.
yarn build
```
